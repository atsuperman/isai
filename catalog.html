Configuration Guide

Firefox Configuration

For the best experience with our platform, we recommend using Firefox with battery optimizations disabled.

1
Open Firefox App Info from your device settings
2
Navigate to Battery settings
3
Select "No Restrictions" mode
4
Restart Firefox for changes to take effect

Benefits: Improved performance, no background restrictions, better battery efficiency

Chrome Configuration

Chrome users should disable battery optimization for the best experience with our platform.

1
Open Chrome App Info in your device settings
2
Go to Battery Optimization
3
Set optimization to "Don't Optimize"
4
Confirm changes and restart Chrome

Benefits: Faster loading times, no background throttling, enhanced stability
