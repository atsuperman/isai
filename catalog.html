📱 For Androids

Chrome Users - Important Notice
🔓 No Restrictions
Ensure full functionality by disabling battery optimizations

📱 Android Settings Guide
1. Open Chrome App Info
2. Navigate to:
Battery → Battery Optimization
3. Select "NO RESTRICTIONS"
4. Confirm changes
